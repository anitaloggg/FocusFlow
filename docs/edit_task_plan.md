# 待辦事項編輯功能實作計畫

這個計畫旨在增加修改已存在任務名稱的功能。

## 擬定變更

### 核心組件

#### [MODIFY] [app.js](file:///c:/Labs/antiDemo/app.js)
- 在 `TodoManager` 類別中新增 `editTask(id)` 方法。

- 編輯邏輯：點擊後彈出提示框輸入新名稱。
